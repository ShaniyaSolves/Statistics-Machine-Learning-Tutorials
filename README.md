
# Astrostatistics Notes & Tutorials

This repository contains a curated set of notebooks based on astrostatistics lectures and supporting materials that I found especially useful while learning statistical and machine learning.

The notebooks walk through core concepts with code, examples, and visualizations:

### Contents

* **01–03:** Probability foundations, descriptive statistics, and sampling methods
* **04–05:** Maximum likelihood estimation and model fitting techniques
* **06:** Hypothesis testing and statistical inference
* **07:** Markov Chain Monte Carlo (MCMC) methods for parameter estimation
* **08:** Introduction to `scikit-learn` and ML workflows
* **09–10:** Density estimation, clustering, and dimensionality reduction (e.g., PCA)
* **11:** Regression methods
* **12–13:** Classification methods including GMM, k-NN, decision trees, and SVMs
* **14:** Neural networks and basic deep learning concepts


### Additional Materials

* `scripts/` contains supporting code used in some notebooks and lectures
